记计算机博弈校内赛

发表于 2017-05-25 更新于 2025-09-06

现在是大三下学期14周的周四，转眼这学期已进入了尾声，这学期做的主要的事情除了魔方机器人就是计算机博弈校内赛了，在14周周一当场答辩之后，校内赛终于可以说是告一段落了，而且最后也是happy ending，在这里记录一下这段宝贵的经历。

简单概括一下这事件，校内赛决赛一共八支队伍，我们队伍质疑了其中五支队伍，其中三支队伍当场退赛，剩下两只队伍在赛后1支成绩降等（主观上退赛），另一支队伍成绩降等，但是取消国赛资格，我们队伍和另一支队伍参加国赛。
